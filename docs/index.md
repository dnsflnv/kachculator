## History
### 1.0.1
First public version. Available calculations: 
- Body Mass Index (BMI);
- A body shape index (ABSI);
- Body fat percentage (BFP);
- Relative fat mass (RFM);
- Body proporyions by McCallum;
- Body proportions by McRobert;
- Repeat maximum (1 RM);
- Cooper test and VO2 max.
